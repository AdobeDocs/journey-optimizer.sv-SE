# innehåller {#contain}

Kontrollerar om den andra argumentsträngen finns i den första argumentsträngen.

## Kategori

Sträng

## Funktionssyntax

`contain(<parameters>)`

## Parametrar

* string

## Signatur och returtyp

`contain(<string>,<string>)`

Returnerar ett booleskt värde.

## Exempel

`contain("rowing is great", "great")`

Returnerar true.
