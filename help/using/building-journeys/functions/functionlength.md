# längd {#length}

Returnerar antalet tecken i stränguttrycket i parametern.

## Kategori

Sträng

## Funktionssyntax

`length(<parameters>)`

## Parameter

* string

## Signatur och returtyp

`length(<string>)`

Returnerar ett heltal.

## Exempel

`length("Hello World")`

Returnerar 11.
