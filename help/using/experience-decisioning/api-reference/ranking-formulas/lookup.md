# Sök efter en  {#list-ranking-formula}

Du kan söka efter en viss rankningsformel genom att göra en GET-förfrågan till Offer Library API som innehåller ID:t i sökvägen till begäran.

**API-format**

```http
GET /{ENDPOINT_PATH}/ranking-formulas/{ID}
```

| Parameter | Beskrivning | Exempel |
| --------- | ----------- | ------- |
| `{ENDPOINT_PATH}` | Slutpunktssökvägen för beständiga API:er. | `https://platform.adobe.io/data/core/dps` |
| `{ID}` | ID:t för den enhet som du vill söka efter. | `rankingFormula1234` |

**Begäran**

```shell
curl -X GET 'https://platform.adobe.io/data/core/dps/ranking-formulas/rankingFormula1234' \
-H 'Accept: *,application/json' \
-H 'Authorization: Bearer {ACCESS_TOKEN}' \
-H 'x-api-key: {API_KEY}' \
-H 'x-gw-ims-org-id: {IMS_ORG}' \
-H 'x-sandbox-name: {SANDBOX_NAME}'
```
