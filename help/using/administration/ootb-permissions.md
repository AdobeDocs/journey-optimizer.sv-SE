# Inbyggda behörigheter {#ootb-permissions}

Inbyggda behörigheter representerar olika behörigheter som kan tilldelas **[!UICONTROL Role]** för att finjustera din användaråtkomst till Journey Optimizer. Behörigheter på hög nivå omfattar behörigheter på låg nivå, vilket beskrivs på [den här sidan](high-low-permissions.md).

| Funktion | Behörigheter |
|-|-|
| AI-assistent | **[!DNL Generate content]**: åtkomst till AI-assistenten för innehållsaccelerator i Journey Optimizer. |
| Kampanjer | **[!DNL Approve & publish Campaigns]**: rätt att godkänna och publicera kampanjer när en princip tillämpas. </br>**[!DNL Manage campaigns]**: läs, skapa, redigera och ta bort kampanjer.</br>**[!DNL Publish campaigns]**: rätt att publicera kampanjer.</br>**[!DNL View campaigns]**: skrivskyddad åtkomst till kampanjer.</br>**[!DNL View campaigns report]**: läs och redigera kampanjrapporter. |

| Datainsamling | **[!DNL Manage datastream]**: läsa, skapa och redigera datastreams.</br>**[!DNL View datastream]**: skrivskyddad åtkomst till datastreams. |
